# Алгоритм расчёта цен скупки золота

Источник истины: Excel-файл. В рабочей папке Excel-файл не найден, поэтому перенос выполнен по расшифровке формул из задачи.

## Главный параметр

```text
Главный курс = B2
```

Если `B2 = 1200`, то `mainRate = 1200`.

## Минимальная цена

Для большинства проб:

```excel
=CEILING((Проба / 583 / 10) * ГлавныйКурс, 0.5) * 10000
```

В JavaScript:

```js
excelCeiling((probe / 583 / 10) * mainRate, 0.5) * 10000
```

Для 585 отдельная формула:

```excel
=ROUND(585/583,5)*ГлавныйКурс*1000
```

## Максимальная цена

583 и 750:

```excel
=IF(1500000 - СтартоваяЦена < 200000, СтартоваяЦена + 200000, 1500000)
```

850, 875, 916, 585:

```excel
=СтартоваяЦена + 200000
```

999:

```excel
=СтартоваяЦена + 150000
```

## Почему 583 базовая проба

583 используется как базовая точка расчёта. Формула `Проба / 583` переводит любую пробу в коэффициент относительно 583. Поэтому 750, 850, 916 и 999 автоматически становятся дороже пропорционально содержанию золота относительно базовой пробы.

## Зачем деление на 10

`Проба / 583` даёт коэффициент около `1.0-1.7`. Умножение сразу на главный курс дало бы слишком крупный промежуточный уровень. Деление на 10 приводит промежуточное значение к масштабу, который потом удобно округляется шагом `0.5` и переводится в сумы через `*10000`.

Пример для 583 при курсе 1200:

```text
(583 / 583 / 10) * 1200 = 120
CEILING(120, 0.5) = 120
120 * 10000 = 1 200 000
```

## Зачем CEILING(..., 0.5)

`CEILING(..., 0.5)` округляет промежуточное значение вверх до ближайшего шага `0.5`. После умножения на `10000` это означает округление стартовой цены вверх с шагом `5000 сум`.

Это важно: цена никогда не округляется вниз.

## Зачем умножение на 10000

После деления на 10 и округления значение находится в условных единицах. `*10000` переводит его в сумы.

Пример:

```text
154.5 * 10000 = 1 545 000 сум
```

## Почему разные правила максимальной цены

Максимальная цена — это не чистая математика пробы, а бизнес-правило:

- 583 и 750 ограничены потолком `1 500 000`, но если стартовая цена слишком близко к потолку, максимум становится `старт + 200 000`.
- 850, 875, 916 и 585 получают стандартный коридор `+200 000`.
- 999 получает более узкий коридор `+150 000`.

То есть минимальная цена считается формулой, а максимальная цена задаёт коммерческий диапазон скупки.
